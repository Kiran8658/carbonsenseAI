CarbonSense is a SaaS platform for MSMEs to measure and reduce carbon emissions
Calculates emissions from electricity and fuel using standard factors
Provides ESG scoring using machine learning models
Forecasts emissions with LSTM-based time series analysis
Detects anomalies using Isolation Forest algorithms
Simulates reduction strategies with Monte Carlo methods
Generates professional compliance-ready reports
Offers AI-driven recommendations for emission reduction
Benchmarks performance against industry standards
Built with FastAPI, Next.js, PostgreSQL, and modern ML tools
